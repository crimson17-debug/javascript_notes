box-sizing : content-box;
The content-box adds the padding as an additional factor
to width and height.
ex: 
{
    width : 100px;
    padding : 10px;
}

the content-box will make the entire width = 100 + 10 + 10 = 120 px


box-sizing : border-box;
the border-box adjusts the padding within the width.
ex:
{
    width : 100px;
    padding : 10px;
}
the entire width is 100 px and padding is adjusted inside the width.


outline : none;
outline : 2px solid red;
| Feature                  | `border`                                  | `outline`                            |
| ------------------------ | ----------------------------------------- | ------------------------------------ |
| **Part of box model**    | ✅ Yes — affects layout and size           | ❌ No — does NOT affect element size  |
| **Position**             | Drawn **inside** the element's dimensions | Drawn **outside** the element's edge |
| **Can be rounded?**      | ✅ Yes (with `border-radius`)              | ❌ No — always rectangular            |
| **Can be set per side?** | ✅ Yes (top, right, bottom, left)          | ❌ No — applies to all sides equally  |



scroll-behavior : smooth;

use this code to understand the root and variables of css
root usually applies globally for the css.
variables of css are used to change the properties without repeation.
we set a variable and we can use it anywhere we want. if we change the root variables values the entire values change.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>CSS Variable Demo</title>
  <style>
    :root {
      --bg-color: #ffffff;
      --text-color: #333333;
      --main-color: #007bff;
      --button-color: #ffffff;
    }

    body {
      background-color: var(--bg-color);
      color: var(--text-color);
      font-family: 'Poppins', sans-serif;
      text-align: center;
      padding: 50px;
    }

    h1 {
      color: var(--main-color);
    }

    button {
      background-color: var(--main-color);
      color: var(--button-color);
      padding: 10px 20px;
      border: none;
      border-radius: 5px;
      cursor: pointer;
      font-size: 16px;
    }

    button:hover {
      background-color: #0056b3;
    }

    .dark-theme {
      --bg-color: #121212;
      --text-color: #eeeeee;
      --main-color: #ff4081;
    }
  </style>
</head>
<body>
  <h1>Welcome to My Themed Page</h1>
  <p>This page uses CSS variables for easy theming.</p>
  <button onclick="toggleTheme()">Toggle Theme</button>

  <script>
    function toggleTheme() {
      document.body.classList.toggle("dark-theme");
    }
  </script>
</body>
</html>
